Solution Overview

Technical Architecture

Modular hybrid-AI architecture integrating Google Cloud and specialized LLMs.

Citizen/Court -> [Kotlin Android App]
  |-- (BNS Query) --> [Google Gemini API]
  |-- (Case PDF) --> [DistilBART via Hugging Face]
  L-- (Data/Auth) --> [Firebase Database]
